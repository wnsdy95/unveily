# Support

unveily is an open-source project under active development.

## Questions and discussions

Use GitHub Discussions for design questions, roadmap discussion, implementation ideas, and general feedback.

## Bugs and feature requests

Use GitHub Issues for reproducible bugs and scoped feature requests. Please avoid sharing private policy text, private URLs, cookies, tokens, or account information in public issues.

## Security issues

Do not open a public issue for vulnerabilities or exploit details. See [SECURITY.md](SECURITY.md) for the security reporting process.
